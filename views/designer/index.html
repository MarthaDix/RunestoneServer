Welcome to the course designers page. You can choose to make a course based on a preexisting book or you can build your own custom textbook from one of the current books. If you are a student or independent learner you do not need to build a course.

Buidling a course allows you to select a book for your own class. Once you have selected and built the book, you will be able to create assignments for your students. These assignments will be publish right in the book. As the instructor, You can grade your students homework, and have access to a dashboard that allows you to monitor student progress.

This page can be used to create custom textbooks to fit your classes needs. To start, log into Github. Then, find the textbook you would like and use the clone button which forks the book into your Github. You can use the left hand panel to view the book in its current state and publish it.
